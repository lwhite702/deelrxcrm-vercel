====================
TASKS: CODIFY & ENFORCE BRANDING

1. CENTRALIZE BRANDING
- Create /src/branding/branding.ts exporting:
  const BRAND = {
    name: "DeelrzCRM",
    tagline: "High Business. High Quality CRM. Powered by Clouds",
    colors: { primary, secondary, background, text },
    assets: {

2. BRANDMARK COMPONENT
- Create /src/branding/BrandMark.tsx:

- Prop: showTagline (default false) to render tagline under logo
- Tailwind dark mode must be enabled in tailwind.config.js (darkMode: "class")

- Find JSX <img> src containing ["logo","deelz","rxcrm","branding/"] (png/svg)
- Replace with:

  <BrandMark className="h-8 w-auto" />
- Preserve className if present; dedupe imports
- Skip src/email/** and src/pdf/** (keep raw <img> there)
