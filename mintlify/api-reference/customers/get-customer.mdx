# Get Customer

Retrieve detailed information about a specific customer.

<Param query="customerId" type="string" required>
The unique identifier for the customer
</Param>

<Param query="include" type="string">
Include related data (orders, payments, interactions)
</Param>

## Request

<CodeGroup>
```bash cURL
curl -X GET \
  -H "Cookie: session=your_session_token" \
  https://api.deelrxcrm.app/api/customers/cust_123
```

```javascript JavaScript
const customer = await crm.customers.get("cust_123");

// Include related data
const customerWithOrders = await crm.customers.get("cust_123", {
  include: "orders,payments"
});
```

```python Python
customer = crm.customers.get("cust_123")

# Include related data
customer_with_orders = crm.customers.get("cust_123", include="orders,payments")
```
</CodeGroup>

## Include Options

Use the `include` parameter to fetch related data:

- `orders` - Include customer's orders
- `payments` - Include customer's payments  
- `interactions` - Include customer interactions
- `multiple` - Use comma-separated values: `orders,payments`

## Error Codes

- `CUSTOMER_NOT_FOUND` - Customer does not exist
- `INSUFFICIENT_PERMISSIONS` - User lacks permission to view customer
- `RATE_LIMIT_EXCEEDED` - Too many requests
