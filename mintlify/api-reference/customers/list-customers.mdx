# List Customers

Retrieve a list of customers with filtering and pagination options.

## Query Parameters

<Param query="limit" type="integer" default="50">
Number of customers to return (max 100)
</Param>

<Param query="offset" type="integer" default="0">
Number of customers to skip
</Param>

<Param query="status" type="string">
Filter by customer status (active, inactive, archived)
</Param>

<Param query="created_after" type="string">
Filter customers created after this date (ISO 8601)
</Param>

<Param query="created_before" type="string">
Filter customers created before this date (ISO 8601)
</Param>

<Param query="search" type="string">
Search customers by name, email, or phone
</Param>

<Param query="sort" type="string" default="created_at">
Sort field (name, email, created_at, updated_at)
</Param>

<Param query="order" type="string" default="desc">
Sort order (asc, desc)
</Param>

## Request

<CodeGroup>
```bash cURL
curl -X GET \
  -H "Cookie: session=your_session_token" \
  "https://api.deelrxcrm.app/api/customers?limit=25&status=active&sort=name&order=asc"
```

```javascript JavaScript
const customers = await crm.customers.list({
  limit: 25,
  status: "active",
  sort: "name",
  order: "asc"
});
```

```python Python
customers = crm.customers.list(
  limit=25,
  status="active",
  sort="name",
  order="asc"
)
```
</CodeGroup>

## Response Fields

<ResponseField name="data" type="array">

    <ResponseField name="id" type="string">

    <ResponseField name="name" type="string">

    <ResponseField name="email" type="string">
    Primary email address

    Primary phone number
    </ResponseField>

    Customer status (active, inactive, archived)
