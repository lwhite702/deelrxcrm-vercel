# Get Team

Retrieve information about a specific team.

<Param query="teamId" type="string" required>
The unique identifier for the team
</Param>

## Request

<CodeGroup>
```bash cURL
curl -X GET \
  -H "Cookie: session=your_session_token" \
  https://api.deelrxcrm.app/api/teams/123
```

```javascript JavaScript
const team = await crm.teams.get("123");
```

```python Python
team = crm.teams.get("123")
```
</CodeGroup>

## Response Fields

<ResponseField name="id" type="string">
Unique identifier for the team
</ResponseField>

<ResponseField name="name" type="string">
Display name of the team
</ResponseField>

<ResponseField name="createdAt" type="string">
ISO 8601 timestamp when the team was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
ISO 8601 timestamp when the team was last updated
</ResponseField>

<ResponseField name="settings" type="object">
Team configuration settings
  <Expandable title="settings properties">
    <ResponseField name="timezone" type="string">
    Team's timezone
    </ResponseField>
    
    <ResponseField name="currency" type="string">
    Default currency code (ISO 4217)
    </ResponseField>
    
    <ResponseField name="locale" type="string">
    Language and region locale
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="subscription" type="object">
Subscription information
  <Expandable title="subscription properties">
    <ResponseField name="plan" type="string">
    Current subscription plan
    </ResponseField>
    
    <ResponseField name="status" type="string">
    Subscription status (active, canceled, past_due)
    </ResponseField>
    
    <ResponseField name="billingCycle" type="string">
    Billing frequency (monthly, annual)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="members" type="array">
Array of team members
  <Expandable title="member properties">
    <ResponseField name="id" type="string">
    User's unique identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
    User's display name
    </ResponseField>
    
    <ResponseField name="email" type="string">
    User's email address
    </ResponseField>
    
    <ResponseField name="role" type="string">
    User's role in the team (owner, manager, staff)
    </ResponseField>
  </Expandable>
</ResponseField>
