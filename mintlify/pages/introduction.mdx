## What is DeelRx CRM?

DeelRx CRM is a modern, full-stack customer relationship management system built with Next.js, TypeScript, and PostgreSQL. It provides businesses with comprehensive tools to manage customers, products, orders, and extended operations like inventory tracking, referral programs, and loyalty systems.

## Architecture

DeelRx CRM is built with modern technologies for scalability and performance:

- **Frontend**: Next.js 15 with TypeScript and Tailwind CSS
- **Backend**: Next.js API routes with comprehensive validation
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: JWT-based with role-based access control
- **Deployment**: Vercel with serverless functions
- **Payments**: Stripe integration for secure transactions

## Multi-tenant Ready

The system is designed from the ground up for multi-tenancy:

- **Team-based Data Isolation**: All data is scoped to teams/organizations
- **Role-based Access Control**: Fine-grained permissions per user
- **Scalable Architecture**: Efficient queries with team-scoped filtering
- **Secure by Design**: Multi-layered security with input validation
